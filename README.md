✅ Todo App

A clean and functional web application for managing your daily tasks.
Add, edit, complete, and delete todos — all with a smooth, interactive UI.

✨ Features

📋 Task Management — add, edit, mark as completed, or delete todos

✅ Bulk Actions — select multiple tasks and apply actions at once

🗑 Clear Options — remove completed or all tasks in a single click

🎨 Modern UI — styled with SCSS, responsive and minimalistic

🖱 Drag & Drop — reorder todos easily by moving them around

🚀 Quick Start
Option 1: Simple Setup (no build tools required)

Download the project files

Open index.html in your browser

Or run a local server:

Python 3

python -m http.server 8000


Python 2

python -m SimpleHTTPServer 8000


Node.js (http-server installed)

npx http-server

Option 2: With Live Server (VSCode users)

Open the project folder in VSCode

Right-click index.html → "Open with Live Server"

🎮 How to Use

✏️ Add a new task with the New button

✔ Mark tasks as completed individually or in bulk

🗑 Delete selected tasks or clear all at once

👁 Toggle visibility of completed todos

🖱 Drag & drop tasks to reorder them

🛠 Technologies Used

HTML5 — structure

SCSS (CSS3) — styling and layout

JavaScript (ES6+) — interactivity and logic

Vanilla JS only — no external frameworks
